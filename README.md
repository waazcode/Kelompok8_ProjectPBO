# ANONIM CAMERA STORE
Proyek ini adalah sistem toko kamera sederhana yang mendukung antarmuka CLI dan GUI Swing. Fungsionalitas utamanya mencakup manajemen akun admin & pelanggan, katalog produk, keranjang belanja, checkout dengan beberapa metode pembayaran (COD, transfer bank dengan pilihan bank, QRIS), serta pelacakan pesanan dari pembuatan hingga persetujuan admin. Produk dan pengguna tersimpan ke file sehingga data bertahan antar sesi. Arsitektur memakai kelas inti seperti Store, Product, Cart, Order, Invoice, dan PaymentMethod, ditambah GUI yang memanfaatkan tabel Swing untuk menampilkan daftar produk, keranjang, dan pesanan secara real-time.
